Deploying jobs.tac.local and events.tac.local
---------------------------------------------
1. Replace all dlls of TAC\libraries with dlls of your instance folder
2. Open solution file and changes one click publish settings to point your instance website folder
3. Build solution and publish the build with one click publish option
4. Install TAC\sc9l_sites.zip package into your sitecore instance
5. Apply web binding on your instance with host names "jobs.tac.local" and "events.tac.local"
6. Add "jobs.tac.local" and "events.tac.local" host names into host configuration (manually or using SIM tool)
7. Lanuch sites and check if all loads fine
8. Enjoy... You have saved all your lab effort :)
